About Veritas AI

Project Overview

Veritas AI is a Machine Learning powered web application that detects whether a news article is Real or Fake using Natural Language Processing techniques. The system transforms news text using TF-IDF Vectorization and classifies it using multiple trained machine learning models.

Artificial Intelligence

Machine Learning based prediction system.

Machine Learning Models

  • Logistic Regression
  • Random Forest
  • K-Nearest Neighbors
  • Neural Network

Technology Stack

  • Python
  • Flask
  • Scikit-Learn
  • HTML5
  • CSS3
  • Bootstrap 5
  • JavaScript

Dataset

The models are trained using labeled news datasets consisting of genuine and fake news articles. The text is preprocessed using NLP techniques before prediction.
